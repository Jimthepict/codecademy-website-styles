  • Ice cream - Used for text against darker background and as the lighter background
  • Burnt Sand - Used for inline links and also text against darker background
  • Orange Eye - Used for active state inline links and other highlights
  • Blae black - Used as the darker background
  • Sludge - Used for text against lighter background

Main Navigation:

  • Buttons styled on skeumorphic design
  • Indicator to show which page the user is currently on - green light simulation
  • On hover state showing indication the link is clickable - text changes to underlined, Orange Eye colour and simulated light starts to flash
  • Active state will also show indication that the link has been clicked - button depressed state shown by means of shifting polarity of internal shadows, & text will change colour to red

Social Icons:

  • The Social Icons are from Font awesome, and styled on the skeumorphic design similar to the main navigation buttons
  • Colours of the icons styled with the Sludge colour, to match main theme
  • On hover state will highlight the colours associated with that particular social network being targeted
  • Active state indicates button press by means of shiftin external shadow

  • Buttons styled with same background as main navigation, with 20 degree skew on background element
  • Text will be coloured with Burnt Sand

Links

  • Links will be styled with an initial colour of Burnt Orange
  • On mouse over, in hover state, cursor will change to pointer and text changes to underlined with Orange Eye colour, opacity will be reduced
  • On-clicking a link, active state, text will change colour to red -
